# Python string example.

Reference: http://www.runoob.com/python/python-strings.html

Python轉義字符(一般)
| Name | Descript |
| :------| ------: |
| \ | 接續(python) |
| \\ | 反斜線 |
| \' | 單引號 |
| \" | 雙引號 |
| \a | 警告 |
| \b | Backspace |
| \e | 轉意 |
| \000 | 空 |
| \n | 換行 |
| \r | 回到行首(光標) |

Python字串運算符號
| Name | Descript |
| :------| ------: |
| + | 接續字串 |
| * | 重複字串 |
| [] | 取出特定索引的字元 |
| [:] | 取出範圍內字元 |
| in | 在字串中 |
| not in | 不存在字串中 |
| r|R | Raw string，原始字串，不考慮轉譯特殊或是不能輸出的字串。原始字串使用方式是在字串前面加上'r|R'。 |
| % | 格式化字串 |
