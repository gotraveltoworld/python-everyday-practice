## 基礎語法

### First Class
```python
"""Declare a one class object.
"""
class FirstClass:
    pass
```
### methods of Class
```python
"""Declare a one class object.
    These are some functions in the class.
"""
class FirstClass:

    def first_method():
        pass

    def second_method():
        pass
```
